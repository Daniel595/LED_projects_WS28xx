Das Projekt "LEDTable" ist das aktuell auf dem Raspberry laufende Java-Projekt zur Entgegennahme der Messwerte/Events.

Das Projekt "desktopMatrixTest" ist ein Programm aus der Anfangsphase der Entwicklung zur Visualisierung der Messwerte. 
Dieses Programm funktioniert nicht mehr, da nun andere/zusätzliche Verzeichnisse genutzt werden.
